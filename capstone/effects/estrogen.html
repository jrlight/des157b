Methods to take Estrogen

Some changes will go away if you stop taking hormones, and some will not. Additionally, hormone therapy will not change some things

Changes that stick:

  • Breast growth
  • Infertility*

Goes away when stopping HRT:

  • Fat redistribution
  • Skin Changes
  • Scent changes

Will not change:

  • Voice
  • Facial and Body hair*
